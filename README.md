# 👨‍💼 Employee Statistics Dashboard

## 📌 Project Overview

This Power BI dashboard provides insights into employee workforce statistics, attrition trends, overtime analysis, salary distribution, and workforce demographics.

## 🎯 Objectives

* Analyze employee attrition trends
* Monitor overtime distribution
* Understand salary band distribution
* Explore employee demographics
* Provide interactive filtering for better decision-making

## 📊 Dashboard Features

✅ Employee Count Analysis
✅ Attrition Analysis
✅ Salary Distribution Analysis
✅ Monthly Employee Trends
✅ Interactive Slicers & Filters
✅ Dynamic KPI Cards

## 🛠️ Tools Used

* 📈 Power BI
* 📄 Excel / CSV Dataset
* 🧮 DAX Measures
* 🔄 Power Query
